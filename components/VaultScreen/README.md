# VaultScreen 目录说明

本目录包含与资产页面相关的多个 React Native 组件集合，负责实现用户资产的展示、管理及相关交互功能。

## 目录结构及主要内容

- `AssetList.js`：资产列表组件，展示用户的资产信息。
- `AssetDetail.js`：资产详情组件，显示单个资产的详细信息。
- `AssetOperations.js`：资产操作组件，提供资产管理相关操作。
- 其他资产页面相关组件，支持资产展示和管理功能。

## 组件职责

- 实现资产页面的 UI 渲染和用户交互
- 管理资产数据的展示和操作流程
- 提供资产相关的交互接口

## 维护建议

- 组件应保持职责单一，便于维护和测试
- 代码风格应统一，遵循项目规范
- 及时更新文档，确保功能说明准确
